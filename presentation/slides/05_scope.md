# PIVP Scope <span class="update-badge">v1.0.4 reframe</span>

## Function refinement, not function expansion

What is in scope for this PIVP application is deliberately narrow. Narrow is correct because the routing core works; what needs validation is calibration, not new functions.

---

<div class="section-divider"><span>What PIVP funds — Function refinement</span></div>

| Focus area | What it means in practice |
|---|---|
| **Intake refinement** | Align question language with how real users in five intake types actually think and speak |
| **Routing logic refinement** | Stress-test the decision-support layer against real user scenarios across Function 1 (commercial) and Function 2 (prosumer) |
| **Proposal clarity** | Make routed outputs readable and credible to a real buyer |
| **Early user validation** | Structured conversations with Function 1 and Function 2 users |
| **Stakeholder proofing** | Formal signal from suppliers, NSCC, Perennia, sector contacts — Function 4 partnership routing depth |
| **NS-specific routing credibility** | Deepen local climate, supplier, and channel context |

---

<div class="section-divider"><span>What is NOT in scope for this application</span></div>

| Out of scope | Why it's out of scope |
|---|---|
| Function 3 (B2B forward-demand) at production scale | Validated as concept; production scale is post-PIVP |
| Function 5 (excess-routing) instrumentation | Pilot-stage in Phase 2; current scope is Function 1 and 2 |
| Mi'kmaw partnership engagement | Held as prospective; awaits institutional introduction (see Mi'kma'ki posture) |
| National expansion | Premature; local proof first |
| Marketplace or directory functionality | Different product category; explicitly disclaimed in landscape |

---

<div class="section-divider"><span>The discipline this requires</span></div>

The five functions exist now in the routing core. They are not all funded for production scale here. Any slide, conversation, or document that presents Function 3 or Function 5 as production-ready is a mistake.

---

> *Scope discipline is credibility. Funders and partners trust founders who know what they are and are not funding this round.*

- Do not say "we also plan to add monitoring" — undercuts the application ask.
