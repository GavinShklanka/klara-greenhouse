# Streamlit Deployment Checklist

If the Streamlit application is inaccessible or throws a "does not exist" error, run through this checklist.

- [ ] Confirm you are logged into the correct Streamlit Cloud account.
- [ ] Confirm the app is connected to the correct GitHub repository: `GavinShklanka/klara-greenhouse`.
- [ ] Confirm the branch is set to `main`.
- [ ] Confirm the Main file path is set to exactly: `presentation/streamlit_app.py`.
- [ ] Confirm the app visibility is set to **Public** if external reviewers need access (Streamlit defaults to private for some repos).
- [ ] Confirm the app opens successfully from the Streamlit Cloud dashboard.
- [ ] Confirm the final, confirmed URL is copied into `README.md`.
